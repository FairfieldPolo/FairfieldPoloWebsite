# About page & sponsors

## Sponsors

Sponsor logos and links on the **About** page (and elsewhere) come from **Sponsor** documents in Sanity.

1. Open **Sponsor** → new or edit.
2. **Sponsor name** — required.
3. **Logo** — upload; set **Alt text** so the logo is described for accessibility.
4. **Website URL** — optional link when visitors click the sponsor.
5. **Sponsorship tier** — title, gold, silver, or supporting (controls ordering and emphasis).
6. **Active seasons** — optional labels such as `2025` if you use them for your own filtering; behavior depends on site implementation.

Publish the sponsor. Logos generally appear in tier order.

## About page story (“Our history,” etc.)

The long narrative text on **`/about`** (history paragraphs, mission copy) is stored in **website source files**, not in Sanity. To change that wording, ask a developer to edit the About page component, or agree on moving that copy into the CMS in a future update.

## Newsletter block on About

The newsletter signup at the bottom of About uses the same backend as everywhere else—see [contact-and-newsletter.md](./contact-and-newsletter.md).
