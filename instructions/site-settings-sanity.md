# Site settings (Sanity)

Studio includes a **Site settings** singleton: one document for club-wide defaults (site name, tagline, logo, hero image, address, phone, email, social URLs, etc.).

## How to edit

1. Open **Site settings** in the Studio sidebar (sometimes under “Structure” depending on configuration).
2. Update fields as needed and **Publish**.

## Relation to the live site

Some fields may be **wired to the front end** immediately; others might be reserved for future features or SEO. If you change something in Site settings and do not see it on the public site, ask your developer whether that field is connected yet—**contact details on the Contact page**, for example, might still be edited in code until the site reads fully from Sanity.

Keeping Site settings **accurate** is still worthwhile as the single source of truth for the club’s official name, logo, and links.
